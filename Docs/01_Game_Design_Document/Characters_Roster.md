# 👾 قائمة الشخصيات والترسانة (Characters Roster)

## 🚀 1. Astro (البطل الأساسي)
* **الوصف:** التميمة الرسمية للمقاومة (GDGOC Banha). مطور شغوف يبحث عن الكود النظيف وسندوتش البطاطس السوري.
* **المركبة:** "The Boilerplate" - مركبة فضائية متوازنة صُممت كبيئة عمل مرنة (Flexible Workspace) قادرة على دمج وتثبيت أي (Open-Source Package) تظهر في ساحة المعركة.
* **مهارة التفادي (Dash):** `Ctrl+Z` - قفزة سريعة لتفادي "الأكواد الميتة" ومقذوفات الأعداء.

## 🔓 شخصيات مستقبلية قابلة للفتح (Unlockable Characters)
*(الشخصيات الإضافية تقدم Base Stats مختلفة وتفضل أسلحة تراكات معينة)*
* **Byte (ممثل الباك إند):** مركبته "The Server Rack". ثقيلة، بطيئة، لكنها تتحمل ضرراً هائلاً (High HP). تتناغم جداً مع أسلحة خوادم الـ Go والـ Node.
* **Pixel (ممثلة الـ UI/UX):** مركبتها "The Canvas". سريعة جداً وحجم تصادمها (Hitbox) صغير. تتناغم بقوة مع أسلحة الـ Alignment والـ Crowd Control.
* **Cipher (ممثل الـ Cybersec):** مركبته "The Proxy". دروع قوية ومهارة اختفاء مؤقت.
