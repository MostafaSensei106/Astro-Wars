# 💥 المؤثرات البصرية والحركية (VFX & Animations)

## ✨ المؤثرات التفاعلية (Feedback)
* **Screen Shake:** اهتزاز الشاشة بشكل طفيف عند تلقي ضربة قوية أو عند استخدام أسلحة הـ Backend الـ AoE.
* **Particle Systems (Flame Particles):**
  * طلقات الـ Flutter تتفتت إلى Widgets صغيرة (مربعات زرقاء) عند الاصطدام.
  * أسلحة الـ Cybersecurity تصنع جدار حماية (Hexagon Grid Shield) يمتص الطلقات بصرياً.
  * الـ Dash يُترك خلفه طيف (Ghost trail) لمركبة اللاعب لإحساس السرعة الفائقة.
